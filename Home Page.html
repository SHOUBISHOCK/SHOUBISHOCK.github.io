Welcome to Redstone! A Nuclear Dawn Community Server!

Pick up games between friday and sunday around 19:00 GMT

Redstone Code of Conduct

  1. Everyone has the right not to be harassed or discriminated against.
  2. Players shall not ruin games, spam/advertise, bomb or use hacks/exploits.
  3. Avoid the usage of advertisements, to intimidate or deceive people.
  4. Commanders shall not violate section i-iii), use build exploits, sell all the spawns or intentionally ruin/throw a populated game.
  5. Frequently stacking on one faction is not acceptable. It strains the quality of games for everyone on the server in pursuit of personal gain.

Click Here - FAQs

What does 100 trick-rate mean?

  • Redstone runs a 100 tickrate to improve responsiveness.
  • The server updates 100 times per second of 66 times.
  • This improves everyone's responsive time by 10ms.

What hardware is this server hosted on?

  • Redstone is hosted on a dedicated i7-4790K with 16GB of ram.
  • Server performance is always 100% at 32 players with this hardware.

ND Bot Features: Allows a high amount of bots with less than 8 players, without effecting team balance.

ND Building Fixes: Destorys a wall when built inside a relay tower or wireless reapeater.

ND Commander Ban: Allows a server admin to ban someone from using commander.

ND Commander Chair: Removes the waiting period for commander applications. Creates a delay to enter the chair instead.

ND Commander Checklist: Creates a task list of important items for new players to complete.

ND Commander Deprioritize: Allows a server admin to deprioritize someone from commanding.

ND Commander Restrictions: Gives experienced commanders first pick, when the server is full of veterans.

ND Commander Tips: Displays a list of commander tips for new player. Only when they are commanding.

ND Gernade Trails: Renders gernade trails for players, spectators and commanders. Enable/Disable via !settings menu.

ND Kill Bots: Allows commanders to type !KillBots in chat, if they are all stuck.

ND Managed Slots: Adjusts the sever slot count based on the map and average player skill.

ND Off Map Fixes: Fixes most exploitable building locations in unfair or unintended locations.

ND Player Skill: Gives each player a skill value. Used to balance teams and enhance server plugins.

ND Player Tips: Triggers tips for new players, when they do something wrong.

ND Pull Bots: Allows players to type !PullBot in chat to unstuck a bot.

ND Radio Controls: Prevents a player who is silenced from spamming radio commands.

ND Resource Messages: Displays how resource generation works, when a player captures a resource point.

ND Resource Spawner: Improves game balance by adding or removing tertaries in select locations.

ND Rock The Vote: Creates a customized !rtv command, to allow players to vote the change the map.

ND Round Restart: Allows players to vote to restart current round, by typing !restart in chat.

ND Set Language: Sets server language automatically based on client ip. Enable/Disable via !settings menu.

ND Spawn Colours: Changes the colour of a player currently under spawn protection.

ND Suicide Controls: Prevents players from absuing the suicide command "becuase it's faster than dieing."

ND Surrender: Allows players to type !surrender in chat to vote to surrender.

ND Team Balance: Automatically places players on team, based on the skill difference.

ND Team Breakdown: Displays on the HUD a breakdown of important troop counts for the team.

ND Team Picking: Allows server admins to activate team-pick mode, where commanders pick teams.

ND Team Shuffle: Automatically shuffles teams at round start, to evenly distribute skill.

ND Team Diff Display: Displays the team difference in small hint text bellow the time.

ND Warmup: Creates a warmup round to give everyone a chance to connect to the server.

Next Map: Allows players to check the next map by typing nextmap in chat.
